## OPOS_Mask_R-CNN
Implementacja algorytmu Mask R-CNN

### Instalacja
1. Wymagane pakiety do uruchomienia kodu określono w pliku requirements.txt

2. Optymalna wersja Pythona to >3.4

## Uruchomienie
```bash
python3 starter.py
```
